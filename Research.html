RESEARCH

  • Correlated Sample Synopsis (CS2): A New Statistic Synopsis for Database Query Optimization.
    • Proposed a novel synopsis, CS2, which retains the join relationships between relational tables in database, and precisely provides estimations for complex SQL queries including joins and selections. Empirically demonstrated using skewed data that CS2 requires much less storage space than any other state-of-the-art synopses and provides considerably more accurate estimation.
    • Introduced a novel statistic estimator called "Reverse Estimator'' (RV Estimator) which conquered the bottleneck of other statistical estimators in the case when no simple random sampling is available. Proved both theoretically and empirically that the RV Estimator is unbiased for given join/select queries.
    • Developed the paralleled algorithm to generate CS2 and implemented the automated query estimation system integrated with MySQL Database.
  • Sample Trace: Optimizing Analytical SQL Queries Using Intermediate Statistics.
    • Proposed a new data structure called "Trace'' which recycles the intermediate information while database engine executes any query. The generated "Trace'' derives the exact query result size for ANY given logical query execution plan.
    • Applied sampling theory in Trace to introduce "Sample Trace'' when extremely reduces space complexity and provides precise query result estimations. In addition, the associated Sample Trace estimator is proved both in theorem and experiments to be unbiased.
  • A Performance Study of High Dimensional Histogram and Sample in Database Query Optimization Context.
    • Built a high dimensional statistic model in which range queries and database space partitions are randomly simulated.
    • Extended the 1-dimensional statistic procedure -- "random shuffling'' into higher dimensional scenarios.
    • Derived the space cost of sample in high dimensional space for range queries and extend the state-of-the-art formula of 1-dimensional histogram space cost to 2-dimensional scenario.
  • Cyber Security

    • Defined the foundation concepts in the feature selection procedure of intrusion detection system.
  • Microarray in bioinformatics
    • Cooperated in the broad-based review of state-of-the-art methodologies in microarray processing technology
    • Summarized the commonly adopted clustering methods in microarray processing

Projects and Software

  • CS2 Query Result Estimation System (CS2-QRES)
    • Implemented the CS2 to work with MySQL database to provide query result estimation using CS2 synopsis. It requires only tiny sized space on disk and can produce query estimations quickly and is capable of easily extended to Big Data environment.
    • Implementation of Tuple Graph Synopses for Relational Data Sets (TuG)
    • Implemented the state-of-the-art TuG database clustering system which adopts BIRCH clustering, Count-Min sketch, CV Tree and etc., using Java and MySQL database. And solved the bottleneck of the large sized intermediate results transportation with SQL procedure and buffer pool.
  • Puppeteer: A CRM System for Cyber Security
    • Implemented using PHP, Shell Script, and MySQL. Deployed on Linux to incorporate with date-to-day enterprise cyber security activities. Already adopted by IT Department of SIUC and in production with fully written project documentations. It centralizes the duties of team members of Information Security Team and responses of LAN Administrators on campus into one system. It also automatically generates emails and reports from data warehouse.
  • Opensource Private Cloud
    • An opensource IaaS and private cloud constructed using OpenNebula Cloud OS from scratch.
      Capable of virtualize Windows/Linux/BSD using KVM/VMWare driver on NFS.
      It flexibly provides virtual machines for cyber security tests, data backups, and etc. In addition, Cianna Cloud saves power and costs for increasing needs of workstations in IT Department.
